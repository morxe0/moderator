### MODERATOR is a general-purpose discord moderation bot written in python3 with the discord.py library.
Moderator is quite simple to set up. Here are the steps:
1. download the newest package.
2. unzip the .zip file.
3. create your bot using https://discord.com/developers/applications.
4. use the setup wizard in your-unzipped-moderator-folder/docs/setup.html

It is unrecommended for you to modify the moderator.py file without knowing what you are doing.
